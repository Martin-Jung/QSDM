Close holes

Brief Description

Closes holes (inner rings) in all patches of the specified landcover class.

Parameters

  • [param] Input landscape raster: Classified raster to work on

  • [param] Landscape class: The value of the pixels that should be removed

  • [param] Output Raster: Output result

Limitations

  • If used on many landcover classes it returns only the chosen class values in a new raster
