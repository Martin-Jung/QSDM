Create Species Richness grid

Brief Description

Generates a new species richness grid based on the number of points in each grid cell. Three different metrics can be computed : Species Richness as sum of unique species per cell. Weighted Endemism, which is the sum of the reciprocal of the total number of cells each species in a grid cell is found in. Emphasizes areas that have a high proportion of animals with restricted ranges. Corrected Weighted Endemism, which is Weighted Endemism divided by Species Richness per grid cell.

Parameters

  • [param] Input Vector Layers: A point layer containing species occurences

  • [param] Input Vector Field: A field containing the name of (individual) species

  • [param] Input Metric selection: User can decide what to calculate

  • [param] Input Grain size: Number indicating the resolution of the new layer. Please note that this value is projection dependent. If you want to make 1Â° resolution grid than your project and layers have to be in this projection too.

  • Output Raster: Results are added to QGIS.
