Niche Overlap Statistics

Brief Description

Calculates for any number of input predictions the quantitative Niche overlap statistics introduced by Warren et al, 2008. This includes the traditional Schoener's D and Warren's I based on Hellinger distances.

Both Schoener's D and Warren's I rank from 0 (niche models have no overlap) to 1 (niche models identical), while the latter treats individual grid cell values as simple probability distributions rather than reflected relative use of a specific grid cell.

Parameters

  • [param] Input Raster Layers: Any number of predicted niche models

  • Output Results: Results are added to QGIS.

Literature

  • Warren, D. L., Glor, R. E. and Turelli, M. (2008), ENVIRONMENTAL NICHE EQUIVALENCY VERSUS CONSERVATISM: QUANTITATIVE APPROACHES TO NICHE EVOLUTION. Evolution, 62: 2868-2883.
